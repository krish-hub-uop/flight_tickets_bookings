# Flight Ticket Booking System

## Project Overview

The Flight Ticket Booking System is a web-based application that allows users to search, book, and manage flights online. The platform offers real-time flight availability, secure payment processing, and a user-friendly interface. Designed for travelers seeking convenience and efficiency, this system also provides robust customer support and user account management.

### Key Features

- **User Accounts**: Secure registration and login, profile management, and booking history.
- **Flight Search**: Comprehensive search functionality with filters for destination, date, price, and more.
- **Real-Time Availability**: Access to live flight data through third-party API integration.
- **Seat Selection**: Users can choose their preferred seats during booking.
- **Secure Payments**: Integration with reliable payment gateways for safe transactions.
- **Booking Management**: Options to view, modify, and cancel bookings easily.
- **Customer Support**: Responsive chat support and an extensive FAQ section.

## Problem Statement

Travelers often face challenges when trying to find and book flights due to a lack of real-time information, complex interfaces, and insecure payment options. This project aims to address these issues by providing:

- **Real-Time Data**: Immediate access to flight availability and pricing, ensuring users can make informed decisions.
- **User-Friendly Interface**: A streamlined booking process that enhances the overall user experience.
- **Secure Transactions**: Implementing robust security measures to protect user data and payment information.
- **Comprehensive Support**: Responsive customer service to assist users throughout their booking journey.

## Technology Stack

- **Frontend**: React.js
- **Backend**: Node.js with Express.js
- **Database**: PostgreSQL
- **Payment Gateway**: Stripe API
- **APIs**: Amadeus Flight API for real-time flight data
